F-THEORY 6D TORIC BASE SURFACES (61,539)
=========================================

Dataset Name: F-theory 6D Toric Base Surfaces

Description:
Complete enumeration of smooth toric base surfaces that support elliptically
fibered Calabi-Yau threefolds for 6-dimensional F-theory compactifications.
Contains 61,539 toric base surfaces described combinatorially.

Source:
